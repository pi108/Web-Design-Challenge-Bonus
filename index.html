Summary: DJIA vs. X

The purpose of this project was to analyze the relationship between the Dow Jones Indstrial Average (DJIA) and the following 4 key global economic indicators (over the period 04/19/2010 to 04/09/2020):
1. Price of Brent Crude Oil
2. Price of Gold
3. The Chinese Yuan to the US Dollar Exchange Rate
4. The Indian Rupee to the US Dollar Exchange Rate

To accomplish this analysis, we first extracted data using an API from the following website:
https://fred.stlouisfed.org/
We then used the Pandas and Matplotlib libraries to clean the data, analyze the data, run regression analyses, and plot the regression graphs for each of the 4 sets of relationships mentioned above, for the entire time-period (2010 to 2020).

These graphs can be seen on the right side of this page. If you click on any of these graphs, it will take you to the page with the details for that particular relationship. The detailed page has a graph for the relationship for the entire time-period (2010-2020), as well as graphs for each of the 11 individual years within that time-period.
If you click on the menu item called "Plots" at the top right hand corner of this page, you can select from each of the 4 sets of relationships and see the graphs for the entire period (2010-2020) for that particular relationship, as well as for each of the individual years for that relationship.
If you click on the menu item called "Comparison" at the top right hand corner of this page, you can see the graphs for the 4 sets of relationships for the entire time period (2010-2020).
If you click on the menu item called "Data" at the top right hand corner of this page, you can see the table with the underlying data that was used for this analysis.
To get back to this summary page from any of the other pages, please click on the home button at the top left of the screen called "Go to Home Page".
